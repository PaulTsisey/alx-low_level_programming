# **C- DEBUGGING**

> *This is an alx full stack engineering course, tasks based on debugging in C programming language.*

<br> - These tasks were done with the vi editor.
<br> - I used the Betty style of coding in all tasks
<br> - All files can be compiled using gcc, using the options > wall > werror > pedantic > std=gnu89
<br> - Various codes were provided to be debugged for this project


### *TASKS*

| task | name | discription |
| ---- | ---- | ----------- |
| 0. | Multiple mains | Based on the file main.c, create a file that test the funtion positive_or_negative() gives the correct output when given a case of 0 |
| 1. | Like, comment, subscribe | Comment out (don't delete it!) the part of the code that is causing the output to go into an infinite loop |
